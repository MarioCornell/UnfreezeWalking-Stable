Scriptable Object Pro - Enhanced Roll-A-Ball - ReadMe

To Build the Roll-A-Ball demonstration

Load the Roll-a-ball Scene from this folder

Open the Build Settings using the menu File->Build Settings

Remove any existing entries under Scenes In Build

Press 'Add Open Scenes'

Select your Platform and press Switch Platform if necessary

Press Build And Run

Note: If you want to persist the data over builds open the script DemoGame->Scripts->SOProObject and remove the comment in front of [PersistAcrossBuilds]
